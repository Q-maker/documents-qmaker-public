# Ouvrir un fichier `.qcm` depuis votre appareil

Un fichier `.qcm` est un fichier de quiz portable. Vous pouvez en recevoir un d'une autre
personne, en télécharger un, en copier un depuis un autre appareil, ou en garder un en dehors de
votre dossier de travail habituel. Quand le fichier n'apparaît pas encore sur l'écran d'accueil,
ouvrez-le depuis l'explorateur de fichiers Android.

Depuis l'accueil, appuyez sur le bouton flottant.

![Bouton flottant sur l'accueil](assets/home-fab-button.jpg)

Dans le menu qui s'ouvre, choisissez l'option **Ouvrir un fichier *.qcm depuis l'explorateur**.

![Option ouvrir un fichier qcm](assets/open-qcm-file-menu-option.jpg)

Android ouvre l'explorateur de fichiers du système. Parcourez les dossiers jusqu'à l'emplacement du
quiz, puis appuyez sur le fichier `.qcm` que vous voulez ouvrir.

![Fichiers QCM dans l'explorateur Android](assets/file-explorer-qcm-list.jpg)

Après la sélection, QcmMaker ouvre le fichier et l'ajoute à la liste de votre espace de travail.
Cela donne aussi à QcmMaker l'autorisation d'accéder à ce fichier sélectionné, pour que vous
puissiez l'ouvrir plus facilement depuis l'accueil la prochaine fois.

Si le fichier n'apparaît pas, vérifiez que vous êtes dans le bon dossier et que le fichier utilise
bien l'extension `.qcm`.
