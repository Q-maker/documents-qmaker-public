# Getting Started

Start here when opening QcmMaker for the first time.

## First Steps

1. [Set up your workspace](workspace-setup.md) so QcmMaker knows where your quiz files live.
2. [Understand the Home screen](interface-overview.md), which acts as your dashboard for opening, playing, editing, sharing, and organizing quizzes.
3. [Create your first quiz](first-quiz.md), add a first question, then test it before sharing.

Good to know: most QcmMaker workflows start from a `.qcm` file in your workspace. If you understand where that file is stored, it becomes easier to edit it, back it up, move it to another device, or send it to someone else.
