# Workspace Setup

On first launch, QcmMaker asks you to choose a workspace. This folder stores and lists your `.qcm` quiz files.

![Workspace onboarding](images/workspace-onboarding.png)

Tap **I AM READY, LET'S GO**, choose the folder you want to use, then tap **USE THIS FOLDER**.

![Android folder picker](images/android-folder-picker.png)

Android asks for final permission. Tap **ALLOW** so QcmMaker can access current and future quiz files in that folder.

![Android folder permission](images/android-folder-permission.png)
