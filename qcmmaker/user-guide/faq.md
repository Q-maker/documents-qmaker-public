# QcmMaker FAQ

## Getting Started

#### What is QcmMaker used for?

QcmMaker is used to create your own quiz files, play them locally, and share them with other people as portable `.qcm` files. A quiz can include questions, answers, pictures, sounds, comments, and automatic scoring.

→ See [Creating your first quiz](getting-started/first-quiz.md) for the full how-to.

---

#### Where are my quizzes stored?

QcmMaker stores quizzes as `.qcm` files in the workspace folder chosen on your device. If a quiz is not visible after changing phones or restoring a backup, look for the old workspace folder or any saved `.qcm` backup, then copy or open the file again on the new device.

→ See [Workspace setup](getting-started/workspace-setup.md) for the full how-to.

---

## Creating Quizzes

#### Can I combine two quizzes into one?

Yes. Open the destination quiz, then use the import feature to import questions from another `.qcm` file into the current quiz.

→ See [Import & Export](features/import-export/README.md) for the full how-to.

---

#### Can QcmMaker create a quiz directly from a PDF or image?

QcmMaker's documented import flow supports existing `.qcm` quiz files and supported text question files. If your source is a PDF or image, prepare the questions in a supported text structure first, or recreate them manually inside the editor.

→ See [Import & Export](features/import-export/README.md) for the full how-to.

---

#### Why are pictures or sounds missing when I play a quiz I just edited?

First make sure you are playing the same quiz file that you edited. If you have several copies of the same quiz in different folders, launch the quiz from the editor with **Save & play using** so the latest saved file is used.

→ See [Play modes](features/play-modes/README.md) for the full how-to.

---

## Playing & Reviewing

#### Can I see the correct answer immediately after answering?

Yes. Use **Challenge mode** when you want immediate feedback during play; use **Exam mode** when you want feedback and correction at the end.

→ See [Play modes](features/play-modes/README.md) for the full how-to.

---

#### Can I review the questions I answered incorrectly?

Yes. After a quiz result, choose **Correct** to review the quiz with answer information and navigation controls. The correction drawer also helps you identify correct, incorrect, and unanswered items.

→ See [Play modes](features/play-modes/README.md) for the full how-to.

---

## Sharing & Compatibility

#### How do I send a quiz to someone else?

Share the quiz as a `.qcm` file from QcmMaker or from your device's file manager. The receiver needs QcmMaker or another compatible QCM reader to open and play that file.

→ See [Sharing](features/sharing/README.md) for the full how-to.

---

#### Where is the code or live link for students to join my quiz?

QcmMaker's normal sharing model is file-based: you share a `.qcm` file, not a live room code. Supported web links can open a hosted quiz in QcmMaker, but a local quiz must first be shared as a file or hosted somewhere that provides a link.

→ See [Open shared quiz links](features/sharing/open-quiz-links.md) for the full how-to.

---

## Premium

#### Will Premium or the trial screen block access to my quizzes?

Premium access controls advanced features, not your saved quiz files. If a trial or Premium screen appears, choose the option that continues the evaluation, restores an existing license, activates a code, or switches to a plan; your quizzes remain stored as `.qcm` files in the workspace.

→ See [Premium features](features/premium/README.md) for the full how-to.

---

#### Why is a feature I paid for still locked?

Make sure you are using the same Google account and the same QcmMaker installation where the purchase was made. If the purchase still does not appear, update QcmMaker, keep the device online, and use the restore or activation option shown in the Premium area.

→ See [Premium features](features/premium/README.md) for the full how-to.

---

#### Are my purchases available after changing phones?

Purchases made through Google Play are linked to the Google account used for the purchase. On a new device, install QcmMaker with the same Google account and use the restore option if the app does not detect the purchase automatically.

→ See [Premium features](features/premium/README.md) for the full how-to.

---

#### What should I do if money was deducted but Premium did not activate?

Check whether the purchase appears in Google Play purchase history and look for a Google Play order number that starts with `GPA`. If Google Play has not confirmed the purchase, QcmMaker cannot activate it automatically; use the order number when contacting support or Google Play.

→ See [Premium features](features/premium/README.md) for the full how-to.

---

## Troubleshooting

#### Why did a quiz disappear from the home screen?

The home screen lists the `.qcm` files found in the current workspace. If a quiz was moved, deleted, stored in another folder, or not included in a phone transfer, locate the `.qcm` file and open it again or add the folder that contains it.

→ See [Interface overview](getting-started/interface-overview.md) for the full how-to.
