# Стандартный TXT-импорт

Используйте стандартный TXT-формат, когда хотите быстро создать вопросы с телефона или компьютера. Вы пишете вопросы в обычном файле `.txt`, сохраняете его, а затем импортируете в QcmMaker.

Это рекомендуемый формат для первого текстового импорта, потому что он сосредоточен на главном: вопросе, правильных ответах, неправильных ответах и необязательном объяснении.

Когда использовать: выберите стандартный формат, если ваш тест в основном текстовый и вы хотите подготовить много вопросов быстрее, чем вводить их по одному в редакторе.

## Базовая структура

Каждая секция вопроса содержит:

| Часть | Как записать |
|---|---|
| Вопрос | Сначала напишите текст вопроса. Он может занимать несколько строк. |
| Правильный ответ | Начните строку ответа с `*`. |
| Неправильный ответ | Начните строку ответа с `-`. |
| Объяснение | Начните строку комментария с `>`. Эта часть необязательна. |

Пример:

```text
Среди следующих животных
какое живет в воде?
-Кошка
-Собака
*Рыба
-Лошадь
> Рыбы являются водными животными.
```

## Добавить несколько вопросов

Разделяйте секции вопросов одной пустой строкой.

```text
Среди следующих животных
какое живет в воде?
-Кошка
-Собака
*Рыба
-Лошадь
> Рыбы являются водными животными.

Какие из этих животных умеют летать?
*Голубь
-Собака
*Ворон
*Орел
> Голуби, вороны и орлы являются птицами.
```

## Краткие правила

- Помещайте вопрос перед ответами.
- Используйте `*` перед каждым правильным ответом.
- Используйте `-` перед каждым неправильным ответом.
- Используйте `>` перед объяснением, когда хотите его добавить.
- Оставляйте одну пустую строку между двумя вопросами.
- Сохраняйте файл как `.txt`.
- Предпочитайте кодировку UTF-8, особенно если файл содержит акценты, символы или текст не на английском языке.

Полезно знать: символы в начале строк ответов сообщают QcmMaker, как интерпретировать файл. `*` означает, что ответ должен считаться правильным, `-` означает неправильный вариант, а `>` становится текстом объяснения, который отображается во время проверки, если он доступен.

## Полезные шаблоны

Вопрос с одним правильным ответом становится простым вопросом с одним вариантом ответа.

```text
Какое расширение файла у теста QcmMaker?
*qcm
-docx
-xls
> Тесты QcmMaker сохраняются как файлы .qcm.
```

Вопрос с несколькими правильными ответами можно использовать для множественного выбора.

```text
QcmMaker позволяет:
*Создавать тесты
*Проходить тесты
*Делиться тестами
-Редактировать электронные таблицы
> QcmMaker создан для создания, прохождения и обмена файлами тестов.
```

Вы можете начать с примера файла здесь: [demo_standard.txt](../../../../resources/qxt/demo_standard.txt).

Когда вы освоите эту структуру, используйте [расширенный TXT-импорт](import-txt-advanced-ru.md), чтобы добавить типы вопросов, медиа и метаданные.
