# Импорт

QcmMaker может импортировать вопросы в существующий тест из другого файла теста `.qcm` или из обычного текстового файла.

Импорт полезен, когда у вас уже есть материалы в другом месте и вы не хотите вручную создавать каждый вопрос заново.

| Источник | Лучше использовать, когда |
|---|---|
| Существующий файл `.qcm` | Вы хотите повторно использовать вопросы из другого теста QcmMaker. |
| Стандартный TXT-файл | Вы хотите быстро написать простые вопросы с телефона или компьютера. |
| Расширенный TXT-файл | Вам нужны типы вопросов, метаданные, URL-адреса медиафайлов или форматированное содержимое. |

На странице информации о проекте откройте **Дополнительные параметры**, затем выберите **Импортировать вопросы**.

![Импортировать вопросы](images/import-questions-dialog.png)

Выберите, является ли источником QCM-файл или текстовый файл.

## Импорт из QCM-файла

Для файла `.qcm` Android откроет средство выбора файлов. Выберите исходный файл, чтобы продолжить импорт.

![Выбор файла для импорта](images/import-qcm-picker.jpg)

Что произойдет дальше: вопросы из выбранного файла будут добавлены в тест, который вы редактируете. Исходный файл останется отдельным файлом.

## Импорт из текстового файла

Вы также можете подготовить вопросы и ответы в файле `.txt`, в том числе на компьютере, а затем импортировать этот файл в QcmMaker.

Если вы только знакомитесь с импортом текста, начните со стандартного формата. Это самый быстрый способ создать готовый к импорту файл с вопросами, ответами и необязательными объяснениями.

| Потребность | Рекомендуемое руководство |
|---|---|
| Быстро создать простой текстовый файл с вопросами и ответами | [Стандартный TXT-импорт](import-txt-standard-ru.md) |
| Добавить типы вопросов, ссылки на медиа, метаданные или форматированное содержимое | [Расширенный TXT-импорт](import-txt-advanced-ru.md) |

Полезно знать: импорт изменяет тест, который вы редактируете. Исходный файл остается отдельным, но импортированные вопросы становятся частью текущего теста.
