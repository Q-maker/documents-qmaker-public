# Challenge Result And Replay

At the end of a Challenge run, QcmMaker shows a result dialog with the score,
elapsed time, and available next actions.

![Challenge result dialog](images/challenge-result-dialog.png)

## Result Actions

The result dialog usually gives three main choices:

- **Replay** starts another Challenge run from the same quiz.
- **Correct** opens the correction view so you can review questions and answers.
- **Leave** closes the player and returns to the previous screen.

## Replay Choices

When several replay options are available, QcmMaker asks what should be replayed.

![Challenge replay options](images/challenge-replay-options.png)

The available choices depend on the quiz and on the last run. Common options
include replaying the whole test, replaying failed questions only, replaying
missed questions only, or replaying bookmarked questions.

Use **Replay failed questions only** when you want focused practice after a
Challenge run. Use **Replay the whole test** when you want a fresh complete
attempt.
