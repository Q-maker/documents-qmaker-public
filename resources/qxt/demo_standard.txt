>Quiz files created from the QuizMaker application have extension as "qcm" for question, choice, multiple.
It is a shareable file format intended to support interactive quiz contents with automatic scoring.
The "qcm" file is like any other file format; it needs a compatible reader in order to be interpreted.
Same as you need MSWord to open a Docx file, MSExcel to open an XLS file or even a PDF reader to read a PDF file, you also need of a "QCM file" reader in order to read the QCM files as did this app and all other QCM files compatible reader.
