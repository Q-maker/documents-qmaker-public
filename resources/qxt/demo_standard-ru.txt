Какое расширение имеют файлы тестов QcmMaker?
*qcm
-exe
-word
-xls
-csv
> Файлы тестов, созданные в приложении QcmMaker, имеют расширение "qcm": question, choice, multiple.
Это переносимый формат файла для интерактивных тестов с автоматической оценкой.
Файл "qcm" похож на любой другой формат файла: для его интерпретации нужен совместимый читатель.
Так же как для открытия файла Docx нужен MS Word, для файла XLS нужен MS Excel, а для PDF нужен PDF-ридер, для чтения QCM-файлов нужен читатель "QCM-файлов", такой как это приложение и другие совместимые читатели QCM-файлов.
