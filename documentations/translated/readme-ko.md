# 알고 계셨나요?

---

### 컴퓨터에서 Q&A를 가져올 수 있습니다
사실 데스크톱 컴퓨터에서 질문과 답변 **(Q&A)** 를 생성하고 이를 Android QuizMaker 편집기로 가져올 수 있습니다.

이를 위해 간단한 텍스트 **(.txt)** 파일을 사용할 수 있습니다.
이 텍스트 파일은 QuizMaker 편집기가 분석하고 이해할 수 있도록 특정 구조를 따라야 합니다.

Q&A 텍스트 파일을 어떻게 구조화해야 하는지 알아보려면 이 [링크][Documentations]를 따라주세요:
[Documentations]

### 또한
QuizMaker 앱을 사용하여 좋은 경험을 했다면 [GooglePlay]에서 앱에 **별 5개 🌟**를 평가하여 우리의 프로젝트를 격려하고 지원하는 것을 주저하지 말아 주세요.

**별 5개는 아마 비용이 들지 않을 수도 있지만**, 우리에게는 각 5개 별이 이 완전히 **무료**이고 **광고 없는** 앱을 영구적으로 개선하기 위해 더 열심히 일하게 하는 **큰** 그리고 **최고의** [격려][GooglePlay]와 [지원][GooglePlay]입니다.
