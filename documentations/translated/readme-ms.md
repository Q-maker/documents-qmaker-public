### Anda boleh mengimport Soal Jawab dari komputer anda
Sebenarnya, anda boleh membuat soalan dan jawapan **(Q&A)** dari komputer meja anda dan mengimportnya ke editor QuizMaker Android anda.

Untuk itu, anda boleh menggunakan fail teks mudah **(.txt)**.
Fail teks ini harus mengikuti struktur tertentu untuk dianalisis dan difahami oleh Editor QuizMaker.

Sila ikut [pautan][Documentations] ini untuk mengetahui cara menyusun fail teks Soal Jawab anda:
[Documentations]
