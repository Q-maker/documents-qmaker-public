### 您可以从计算机导入问答
实际上，您可以从台式计算机创建问题和答案**(Q&A)**并将其导入到您的Android QuizMaker编辑器。

为此，您可以使用一个简单的文本**(*.txt)** 文件。
此文本文件应遵循特定结构，以便QuizMaker编辑器可以解析和理解。

请按照此[链接][Documentations]了解如何构建您的Q&A文本文件：
