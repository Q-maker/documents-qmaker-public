# Bookmarks

Bookmarks group questions you want to review later.

Open the Home drawer, then choose **Bookmarks**.

![Bookmark activity](images/bookmark-activity.png)

The list includes system bookmarks such as skipped questions and weakness points. Tap a bookmark to inspect or replay its questions when available.

Use the add action to create a custom bookmark.

![Add bookmark](images/bookmark-add-dialog.png)
