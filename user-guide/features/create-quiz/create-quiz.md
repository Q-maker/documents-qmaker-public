# Create A Quiz

Open the workspace actions menu from Home, then tap **Create a new quiz project**.

![Workspace actions menu](../../getting-started/images/workspace-actions-menu.png)

The help icon explains that the new project is saved as a `.qcm` file.

![Create quiz help](../../getting-started/images/dialog-create-quiz-help.png)

Enter a name and tap **CREATE**.

![Create quiz dialog](images/dialog-create-quiz.png)
